Utilizando como database uma tabela de dados declarada no próprio corpo da página, uma pequena rotina em javascript é adicionada na seção de script deste arquivo de modo a automatizar a procura de dados específicos nesta tabela pelo método de filtragem.

Recurso

Uma forma de exibir PDF's é criando um link para o arquivo armazenado em folder específico, como é mostrado a partir deste link, o qual abre o arquivo na mesma janela. Este abre o arquivo em uma nova janela.
