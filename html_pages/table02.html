Introdução

Utilizando como database uma tabela de dados declarada no próprio corpo da página, uma pequena rotina em javascript é adicionada na seção de script deste arquivo de modo a classificar os dados declarados nesta tabela em ordem alfabética ou numérica.

Fonte: StackOverflow.
